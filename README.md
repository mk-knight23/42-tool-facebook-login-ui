# Facebook Login UI - High-Fidelity Angular Clone

A professional, pixel-perfect clone of the modern Facebook Login experience, engineered with Angular 19 and Tailwind CSS. This project demonstrates high-fidelity UI replication combined with robust functional simulation.

## Overview
This repository serves as a technical showcase for building advanced authentication interfaces. It leverages modern Angular features like Signals and Reactive Forms to create a responsive, accessible, and high-performance login flow simulation.

## Features Comparison

| Feature | Legacy Version | Upgraded (v2.0) |
| :--- | :--- | :--- |
| **Tech Stack** | Basic HTML/CSS | **Angular 19 + TypeScript + Tailwind** |
| **Validation** | Minimal | **Strict Reactive Form Validation** |
| **UX** | Static | **Interactive Simulation with Loading States** |
| **Design** | Basic | **Pixel-perfect 2026 FBDS Replication** |
| **Theming** | Light Only | **Automatic Dark/Light Mode Support** |
| **Accessibility** | Limited | **ARIA-compliant & Screen Reader Optimized** |

## Tech Stack
- **Framework:** Angular 19 (Standalone architecture)
- **Styling:** Tailwind CSS (Custom Facebook palette)
- **State:** Angular Signals (UI state)
- **Logic:** Reactive Forms (Auth validation)
- **Icons:** Lucide Angular

## Project Structure

## Setup & Build Instructions

### Prerequisites
- Node.js 18.x or higher
- npm 10.x or higher

### Installation
```bash
# Install dependencies
npm install
```

### Development
```bash
# Start development server
npm start
```

### Production Build
```bash
# Build for production
npm run build
```

Optimized for static deployment on Vercel, Netlify, or GitHub Pages.
